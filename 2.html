＃CSS 선택자(방문했던 링크 색을 바꾸는 법)

1. 태그 선택자 : 가장 포괄적이다 == 가장 힘이 약하다
2. 클래스 선택자 : class="이름1"로 지정을 하고 head에서 .이름1 으로 불러낸다.
3. 아이디 선택자 : 가장 구체적이다 == 가장 힘이 세다. id="이름2"로 지정을 하고 head에서 #이름2 으로 불러낸다.

＃CSS 박스모델

<head> <style> /* block level element */ h1{ border-width:5px; border-color:red; border-style:solid; display:inline; } /* inline element */ a{ border-width:5px; border-color:red; border-style:solid; display: block; } </style> </head>
보다시피 h1 태그는 화면 전체를 쓰고(==block level element), a 태그는 그 텍스트만큼의 크기만 쓴다.(==inline element)
display:inline 또는 display:block 코드를 추가하면 사용 크기를 바꿀 수 있다.
중복을 제거하면, <head> <style> h1,a{ border:5px solid red; } </style> </head> 이런 코드가 된다. 선택자에 콤마(,)를 활용하고 안에는 순서 관계없이 띄어쓰기로 구분 가능.

이와 같이 박스 영역에 각각 이름이 있는데 padding:20px; 처럼 크기를 조절하는데 유용하다.
또한 웹페이지에서 우클릭하면 검사(inspect)를 볼 수 있는데 현재 CSS boxmodel의 상세값을 알 수 있다.

＃CSS 그리드

<div>태그는 어떠한 특성도 없는 태그이다. block level element이다. (<span>태그도 특성이 없는데, inline element이다.)
1.그리드를 사용하고자 하는 내용에 부모 태그를 설정한다.
<div> <div>NAVIGATION</div> <div>ARTICLE</div> </div>
2.부모 태그에 id선택자를 부여한다.
<div id="grid"> #grid{}</div>
3.선언에
display:grid; grid-template-columns: 150px 1fr; 코드를 작성한다.
여기서 150px은 고정, 1fr 크기가 화면 비율에 따라 바뀐다.

＃반응형 디자인, 미디어 쿼리

웹페이지의 크기에 따라 보여지는 게 다르게 하기 위한 방법 == mediaquery
웹페이지 우클릭해서 검사하면 페이지 크기의 픽셀값을 알 수 있다.
@media(max-width:800px){ div{ display:none; } }
@media 코드를 작성한다.
max(화면 크기가 800px보다 작아지면 div내용 display가 none.)는 최댓값, min(800px보다 커지면 display가 none.)이 최솟값이다.

＃CSS 코드의 재사용

다른 웹페이지에 CSS코드 복붙하느라 힘드셨죠? 방법 알려드림 1. style태그 안에 있던 css 코드만 빼서 (style 태그 제외) 새로운 파일(style.css)을 만든다. 2. head 태그 안에 <link rel="stylesheet" href="style.css">코드를 작성한다. 여기도 style 태그는 안 쓴다. 이제 1억개의 웹페이지가 있을 때 css 수정은 style.css 파일에서만 하면 된다.
