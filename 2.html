style을 작성해주어 CSS 코드임을 알리는게 중요한 약속이다. 여기서 style=""은 HTML의 속성이다. 따옴표 안은 CSS의 효과가 들어온다.
a{}의 a는 웹페이지의 모든 a태그를 선택한다고 해서 선택자(selector). 그 안은 선언(Declaration), color는 Property, red는 Value

＃CSS 선택자(방문했던 링크 색을 바꾸는 법)

1. 태그 선택자 : 가장 포괄적이다 == 가장 힘이 약하다
2. 클래스 선택자 : class="이름1"로 지정을 하고 head에서 .이름1 으로 불러낸다.
3. 아이디 선택자 : 가장 구체적이다 == 가장 힘이 세다. id="이름2"로 지정을 하고 head에서 #이름2 으로 불러낸다.

＃CSS 박스모델

<head> <style> /* block level element */ h1{ border-width:5px; border-color:red; border-style:solid; display:inline; } /* inline element */ a{ border-width:5px; border-color:red; border-style:solid; display: block; } </style> </head>
보다시피 h1 태그는 화면 전체를 쓰고(==block level element), a 태그는 그 텍스트만큼의 크기만 쓴다.(==inline element)
display:inline 또는 display:block 코드를 추가하면 사용 크기를 바꿀 수 있다.
중복을 제거하면, <head> <style> h1,a{ border:5px solid red; } </style> </head> 이런 코드가 된다. 선택자에 콤마(,)를 활용하고 안에는 순서 관계없이 띄어쓰기로 구분 가능.

이와 같이 박스 영역에 각각 이름이 있는데 padding:20px; 처럼 크기를 조절하는데 유용하다.
또한 웹페이지에서 우클릭하면 검사(inspect)를 볼 수 있는데 현재 CSS boxmodel의 상세값을 알 수 있다.
