＃함수

(1)함수 기본 문법 : function 함수이름(){ 실행할코드; } 그리고 함수이름(); 을 입력하면 함수가 실행(출력)된다.

(2)매개변수(Parameter)와 인자(Argument) <script> function sum(left,right){ document.write(left+right+'<br>'); } sum(2,3); sum(3,4); </script> left와 right가 매개변수, sum안에 2,3,4가 인자이다.

(3) 리턴(Return) <script> function sum2(left,right){ return left+right; } document.write(sum2(2,3)+'<br>'); document.write('<div style="color:red">'+sum2(2,3)+'</div>'); document.write('<div style="font-size:3rem;">'+sum2(2,3)+'</div>'); </script> 함수의 매개변수로 입력된 값을 리턴으로 출력하면서 함수를 다양하게 활용할 수 있게 되었다.

UI vs API

User Interface : 유저가 특정 조작장치를 사용해 시스템에 관여하는 것
Application Programming Interface : 프로그래밍 언어가 제공하는 기능을 사용하게 해주는 인터페이스
UI 뿐만 아니라 API도 알게 되었다!
